Manual de Ejecución Absoluto

Selecciona tu sistema operativo para ver las instrucciones exactas paso a paso de cómo levantar toda la arquitectura de esta aplicación.

Levantar el Backend (Servidor Local + Túnel)

Abre tu terminal (CMD, PowerShell o Cmder) en la carpeta raíz del proyecto y ejecuta el orquestador:

npm run start-global
Levantar el Frontend (Esta pantalla)

Abre una SEGUNDA terminal en la misma carpeta raíz y enciende el servidor web local:

npm run serve
Abre tu navegador obligatoriamente en: http://127.0.0.1:8080/base.html (Si no le pones base.html abrirá el index).
Compilar hacia Android

Cuando tu código esté listo, en otra terminal transfiere el código a la "caja fuerte" nativa y compila:

npx cap sync
cd android
construir.bat
En Linux VPS no usamos npm run serve ni Tailscale, todo se ejecuta desde los binarios compilados de PocketBase para Producción.
Dar Permisos y Ejecutar (Público)

Sube tu PocketBase para Linux mediante SFTP (FileZilla). Entra por SSH (Consola) y dótalo de vida:

chmod +x pocketbase
./pocketbase serve --http="0.0.0.0:80"
Hacerlo Inmortal (Background)

Si cierras SSH, se apagará. Usa screen para dejarlo corriendo eternamente en segundo plano:

screen -S mi-backend
./pocketbase serve --http="0.0.0.0:80"
# PRESIONA: Ctrl + A, y luego la tecla D (Para salir y dejarlo vivo)
Conectar tu App Frontend

En tu código local en la PC, abre el CPanel (Modo Dios), selecciona el entorno PROD, y escribe la IP de tu VPS Linux (Ej: http://198.51.100.25). ¡Luego compila la app y listo!
